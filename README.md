## Istrukcja obsługi
1. Zainstaluj Node js
2. Sklonuj repozytorium np. za pomocą Gita:
`git clone git@github.com:Zdidhu112/Boty.git`
3. Zainstaluj muduły
`npm i`
4. Uruchom aplikację
`node app.js`
## Warianty:
- **app.js** - uproszczona wersja, wyłącznie do formularzy jednostronicowych.
- **app2.js** - obsługuje formularze z wieloma sekcjami.
